BigyanCore CBT v2.0

An interactive Chemistry Computer-Based Test (CBT) platform for Class 11 & 12 students built with HTML, CSS, JavaScript and Firebase.

Features:
- Google Authentication
- Student Dashboard
- Performance Analytics
- Firestore Result Tracking
- Chapter-wise CBTs
- Responsive UI

Roadmap:
- More Atomic Structure CBTs
- Some Basic Concepts of Chemistry
- Admin Dashboard
- Mock Test Series
