# 🧪 BigyanCore Learning

> Learn Science the Smart Way.

BigyanCore Learning is a modern web-based Computer Based Test (CBT) platform designed for Class 11 & 12 Science students. It provides chapter-wise study resources, practice questions, previous year questions, analytics, and real exam-like CBTs.

---

## ✨ Current Features

### 🔐 Authentication
- Google Sign-In (Firebase Authentication)
- Persistent Login
- Secure Logout

### 👤 Student Dashboard
- Personalized Welcome
- Continue Learning
- Recent Activity
- Performance Summary
- Quiz Analytics
- Best Score
- Average Score
- Questions Solved
- Study Time

### 📝 CBT Engine
- Real Exam Interface
- Timer
- Multiple CBTs per Chapter
- Auto Result Calculation
- Firestore Result Storage

### 📊 Analytics
- Chapter Progress
- Best Performance
- Average Score
- Recent Attempts
- Continue Learning System

### 📚 Chapter System
- JSON-driven Chapter Pages
- Reusable Chapter Template
- CBT Statistics
- Notes / Practice / PYQ Sections
